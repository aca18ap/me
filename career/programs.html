These are the projects I have enjoyed the most during my four years at university

Master's Project

Ecommerce website created from scratch as my master's year project. A Lean startup model was used to come up with a product, various Agile methodologies explored to collaborate with a team of five. I was a heavy influence throughout the whole project, with my main contribution being the management of products, he suggestion system and the design of most of the UI.

Ruby on Rails, HTML, CSS, Javascript
Bachelor Dissertation

My dissertation consisted in studying and upgrading an existing dashboard that interfaces an Aquaponics IoT device. An extensive research behind UI/UX, Colors and Dashboards was carried out as reported in the final thesis. Alongside the improvements to the dashboard itself, I have also created an emulator to test out the dashboard without the physical device.

Dashboard Emulator
Node.js, Vue.js, Python
Android Development

This project involved the creation of an images tracker for Android Devices. Developed using the MVVM design pattern, I was mainly responsible for developing the interface and the various pages using Jetpack navigation, as well as displaying the data correctly throught the MVVM pattern.

Kotlin, Android Studio
Document Retrieval

The main goal of this project was the retrieval of the most relevant documents based on a series of queries. The techniques involved use the bag of words model, using term frequencies and inverted term indices to weight the terms and create vectors to be compared for similarity.

Python
Image mosaic with GPU

The main goal was to parallelize a task originally ran serially by the CPU. This included using openMP to parallelize the task within the CPU and then using CUDA to utilize the power of Nvidia GPUs to run the program in parallel.

C, openMP, CUDA
Internet of Things

Two projects involving the use of an ESP32 controller and various attachments. The first is a morse code writing device, using a spoon as input device a user can connect to the AP generated and visualize what's written. The second more interesting project, involved a weight sensor to measure the amount of loo roll left in a basket, when below a certain limit this would notify the user (hardware sadly unfinished).

Morse Loo Roll
C, C++, ESP32
3D Graphics

This project involved recreating a three-dimensional scene using OpenGL. This was programmed through the Java extension Jogl. The scene is animated, with an interface that allows the user to interact with the scene

Java, OpenGL, JOGL
Web Development with Node.js

This project required to apply various modern concepts to future-proof the way we coded. The project consists in a chat system with various functions. Node.js, Javascript
